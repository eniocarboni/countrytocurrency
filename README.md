**countrytocurrency** is a javascript class to select default currency by browser .

## How to use
```
  <html>
    <body>
    <script src="country_currency.js" ></script>
    <script>
      var c=new DefaultCurrency();
      console.log(c);
    </script>
    </body>
  </html>
```
See test/test.html
